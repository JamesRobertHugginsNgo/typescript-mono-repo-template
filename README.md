Typescript mono repo project template.

# Usage

## Clone

```
git clone https://github.com/JamesRobertHugginsNgo/typescript-mono-repo-template.git PROJECT_NAME
```

## Remove Git Folder

MacOS/Linux

```
cd PROJECT_NAME
rm -rf .git
```

Windows

```
cd PROJECT_NAME
rmdir /s /q .git
```
